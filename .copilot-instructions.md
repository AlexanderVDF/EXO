# EXO — Instructions Copilot

## Projet
EXO v4.2 — Assistant vocal local (C++17 / Qt 6.9.3 / CMake / Windows).
Pipeline : capture micro → VAD → STT → NLU → Claude API → TTS → sortie audio.

## Contrainte Anti-Piper
- Ne JAMAIS proposer, recommander, ni mentionner Piper TTS.
- Utiliser "moteur TTS" ou "backend TTS" comme termes génériques.
- Le TTS d'EXO est XTTS v2 (Coqui) avec fallback Qt TextToSpeech (Microsoft Julie).

## Stack technique
- **Langage** : C++17, Python 3.11 (STT/TTS) / Python 3.13 (backend HA)
- **Framework** : Qt 6.9.3 (msvc2022_64), CMake
- **TTS** : XTTS v2 via tts_server.py (port 8767), 24000 Hz mono PCM16
- **STT** : whisper.cpp + Vulkan GPU (port 8769) / faster-whisper CPU fallback (port 8766)
- **AI** : Claude claude-sonnet-4-20250514, SSE streaming, 8 Function Calling tools
- **Audio** : QAudioSink 24000 Hz mono Int16, RtAudio WASAPI (capture)
- **GUI** : QML VS Code style + React admin
