🎤 1) Capture + DSP (C++/Qt)
────────────────────────────────────────
Créer une classe VoicePipeline (ou refondre VoiceManager) :

- DSP prétraitement :
  - High‑pass 150–200 Hz
  - Normalisation RMS
  - Noise gate léger
  - Anti‑clipping
  - AGC optionnel
- Buffer circulaire
- Signaux Qt :
  - audioLevelChanged
  - speechStarted
  - speechEnded
  - partialTranscript
  - finalTranscript

🧠 2) VAD neuronal Silero (ONNX)
────────────────────────────────────────
Intégrer Silero VAD via ONNX Runtime (CPU).

- Fenêtres 20–30 ms
- Score voix ∈ [0,1]
- Début parole : score > seuil_start
- Fin parole : score < seuil_stop
- Durée min/max utterance
- Adaptation bruit ambiant

Signaux Qt :
- speechStarted()
- speechEnded()

🎧 3) Whisper large‑v3 (streaming, GPU)
────────────────────────────────────────
Créer un service Python `stt_server.py` utilisant faster‑whisper.

IMPORTANT — ACCÉLÉRATION GPU OBLIGATOIRE :

from faster_whisper import WhisperModel
model = WhisperModel(
    "large-v3",
    device="cuda",          # GPU obligatoire si disponible
    compute_type="float16"  # FP16 pour vitesse maximale
)

Fallback automatique :
device="cpu", compute_type="int8"

Fonctionnement :
- réception chunks PCM16
- transcription streaming
- envoi :
  - { "type": "partial", "text": "..." }
  - { "type": "final", "text": "...", "segments": [...] }

Côté C++ :
- partialTranscript(QString)
- finalTranscript(QString)
